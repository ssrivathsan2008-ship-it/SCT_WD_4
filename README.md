# ZenTodo — Premium To-Do Web App

A gorgeous, highly responsive, single-page To-Do web application built using modern HTML5, Vanilla CSS, and Vanilla JavaScript. It offers a smooth glassmorphism dark mode, interactive progress tracking, multi-list organization, priority highlighting, date/time scheduling, and data persistence via `localStorage`.

## Features
- 🗂️ **List Collections**: Group and isolate tasks by project, interest, or context (e.g. Work, Personal, Fitness).
- 🕒 **Due Date & Time**: Set dates and times for tasks. Overdue items are automatically highlighted in red.
- ⚡ **Priority Matrix**: Color-coded Low, Medium, and High priority configurations.
- 🔄 **Sort & Search**: Dynamically filter tasks by status (All, Active, Completed), sort them by various options (Due Date, Priority, Title), and perform fuzzy search.
- 📝 **Inline & Modal Editing**: Edit task attributes (title, date/time, priority, list category) using a sleek popup overlay.
- 💾 **Data Persistence**: Automatic saving to browser local storage.
- 📱 **Fully Responsive**: Adapts seamlessly to mobile, tablet, and desktop monitors.

## Installation & Usage
Since this is a client-side vanilla web app, there are no dependencies to install.

1. Open `index.html` directly in any modern browser, or use a local development server like VS Code Live Server or python:
   ```bash
   python -m http.server 8000
   ```
2. Navigate to `http://localhost:8000` to interact with the application.
